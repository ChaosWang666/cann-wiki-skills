# Wiki Skills

Skills for AscendC Kernel Wiki knowledge retrieval and session feedback.

| Skill | Purpose |
|-------|---------|
| ascendc-ask | Human-facing wiki search via MCP, auto-fetch top-3, synthesize cited answers (renamed from `wiki-query` to avoid collision with the MCP server-side dispatcher of the same legacy name) |
| session-upload | Upload session transcript for experience extraction |

Both require MCP Server running with wiki_search, wiki_get_page, wiki_submit_trajectory tools.
