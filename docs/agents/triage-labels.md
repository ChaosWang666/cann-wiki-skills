# Triage Labels

Skill 中提到的五个规范化 triage 角色，与本仓 issue tracker 实际使用的 label 字符串的对应关系。

| `mattpocock/skills` 中的 label | 本仓 tracker 中的 label | 含义                              |
| ------------------------------ | ----------------------- | --------------------------------- |
| `needs-triage`                 | `needs-triage`          | 需要 maintainer 评估              |
| `needs-info`                   | `needs-info`            | 等待 reporter 补充信息            |
| `ready-for-agent`              | `ready-for-agent`       | 规格明确，可交由 AFK agent 处理   |
| `ready-for-human`              | `ready-for-human`       | 需要人工实现                      |
| `wontfix`                      | `wontfix`               | 不会处理                          |

当 skill 提到某个角色时（例如"打上 AFK-ready 的 triage label"），使用上表右列的 label 字符串。

要改成自己实际使用的词汇表，编辑右列即可。
